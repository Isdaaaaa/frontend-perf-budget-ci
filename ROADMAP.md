# Roadmap

1. **Bootstrap** — Repo setup, lint/test tooling, CI skeleton.
2. **Ingestion** — Parse Lighthouse and bundle stats artifacts; persist baselines.
3. **Budgets Engine** — Compute deltas and pass/fail across core metrics.
4. **PR Reporting** — Render concise GitHub comments with offenders and quick fixes.
5. **Dashboard** — Show recent runs, trends, and budget history.
6. **GitHub App Hardening** — Auth/webhook handling, example Action workflow, and demo content.
